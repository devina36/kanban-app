## Simple Kanban Board

Simple Kanban Board and the datas using API. The tech using React.js and to manage states, using Redux. Also styled using TailwindCss

## Features

- CRUD task
- Create Todos
- Login
- SignUp

## Tech Stack

- React
- Vite
- React Redux
- Tailwind Css
- React Icons
